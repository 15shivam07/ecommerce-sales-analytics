# Methodology

## Project Workflow

This project followed a structured end-to-end analytics process to transform raw sales data into actionable business insights.

---

## 1. Data Collection

- Source: Superstore Dataset
- Total Records: 9,800
- Imported into MySQL database for analysis.

---

## 2. Data Validation

The dataset was reviewed for:

- Missing values
- Duplicate records
- Incorrect data types
- Revenue consistency checks

Import issues encountered during MySQL ingestion were investigated and resolved using LOCAL INFILE.

---

## 3. SQL Analysis

Business questions were answered using SQL.

Key analysis areas included:

### Customer Analysis

- Total customer count
- Revenue contribution by customer
- Customer concentration analysis

### Geographic Analysis

- Revenue by state
- Revenue by region
- Top-performing states

### Product Analysis

- Category performance
- Sub-category performance
- Regional category mix

---

## 4. Power BI Dashboard Development

Insights from SQL analysis were translated into interactive dashboards.

Dashboard Pages:

### Executive Overview

- Revenue KPIs
- Orders KPIs
- Customer KPIs
- Category Performance
- Regional Revenue Trends

### Customer & Geographic Analysis

- Customer concentration analysis
- State-level revenue analysis
- Geographic revenue distribution

### Product & Growth Analysis

- Category mix by region
- Technology sub-category performance
- Monthly revenue trend analysis

---

## 5. Business Insight Generation

Findings were converted into actionable recommendations focusing on:

- Revenue concentration risk
- Geographic opportunities
- Product growth opportunities
- Customer diversification strategies
